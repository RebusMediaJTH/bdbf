• Select Topics Menu and Course Menu buttons to go to the Course Menu or the Topics Menu.
  • The Back Button takes you to the previous page of the course. Note that during an assessment, you cannot go back through the questions, so make sure you're happy with your answer before selecting Next.
  • When active, the Next button takes you to the next page of the course, and on the last page of a topic, it'll return you to the Menu page.
  • Select the Listen icon if you prefer to listen instead of reading content that is presented in text.
  • Select the Read icon if you prefer reading instead of listening to the content that is presented in audio and video voice overs.
  • To go to the next page in the course select the button, at the bottom of the page, that says 'Continue' or 'Yep, got it'.
  • The Save & Exit button allows you to exit the course at any time. Your progress will be saved so that next time you enter, you can return to where you left off.
